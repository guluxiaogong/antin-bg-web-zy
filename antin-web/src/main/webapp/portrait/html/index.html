资金来源

    说明：根据市民历史的就诊付费类型进行统计，对付费类型进行排列，根据标签顺序进行标记。
    标签：自费病人、医保病人、公务员报销。
    数据来源：门诊费用表-付费情况-历次就诊中占比大于50%的标签。

    慢性病类型

    说明：根据市民的历史诊断信息，由于慢性病是需要长期的往返医院进行复查，并且持续性较久的疾病类型，多个慢性病存在并发的可能性，通过对ICD10中慢性病的种类进行比对，对患病市民进行标记。
    标签：高血压、糖尿病、癫痫、痛风、心血管疾病等。

    用药依从性分类

    说明：对于慢性疾病或者伴随性疾病可能需要长期不间断的服用某种药品，此处对长期用药的患者进行识别，并标记信息。
    标签：长期用药、无依从。
    数据来源：门诊用药记录。

    用药习惯分类

    说明：根据患者的历史就诊信息，对药品分类进行统计，分析出患者的用药习惯。
    标签：西药、中药等。
    数据来源：门诊用药记录-用药类别。

    就医习惯

    说明：通过对市民的历史就诊记录进行统计，分析市民经常前往就诊的医疗机构，通过该项标签可以识别出患者对于三级医院或基层医疗卫生机构的依赖性。
    标签：厦门大学附属第一医院、嘉莲社区卫生服务中心等。
    数据来源：门诊就诊基本信息。

    长期用药（慢性病）

    说明：对于慢性疾病或者伴随性疾病可能需要长期不间断的服用某种药品，例如糖尿病患者根据严重程度不同可能需要按时注射胰岛素或服用适量的糖尿病药物。
    标签：二型胰岛素（注射）、西格列汀二甲双胍片、阿卡波糖片、亚莫利格列美脲片等。

    传染病分类

    说明：根据历史统计的传染病信息进行标记。
    标签：肺结核、HIV等。
    备注：传染病信息需注意患者隐私。

    遗传病分类

    说明：根据历史统计的遗传病信息进行标记。
    标签：糖尿病、癫痫等。

    生物学分类

    说明：基于人口学属性和体征信息进行分类标记。
    标签：
        年龄（新生儿、婴幼儿、学龄前儿童、儿童、青少年、中年、老年等）、
        体重（消瘦、偏瘦、适中、偏胖、肥胖），
        婚姻（已婚、未婚），
        是否有小孩（宝妈、宝爸）、
        小孩个数（1、2、3个）、
        职业（程序员等），
        性别（男性、女性），
        人种（黄、白、黑），
        民族（满、汉）、
        籍贯（黑龙江、福建），
        血型（A型血、B型血等）。
    数据来源：个人基本信息、体检。
